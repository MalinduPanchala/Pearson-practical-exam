CRUD application for manage countries

App made by Malindu Panchala

This app was made for the practical exam from Pearson lanka for applying to an internship

The frontend of the application is developed with Angular framework and Material Design library and the backend is developed with Java spring boot framework

API base url is "http://localhost:8080/rest/v2/" or "http://127.0.0.1:8080/rest/v2/countries/" and POST,GET,PUT,DELETE can be requested with "countries/" suffix to the base url
